# MPI Πρόγραμμα - AskisiI

Το παρόν MPI πρόγραμμα (σε γλώσσα **C** και θεωρώντας point-to-point communication) αποσκοπεί στην επίδειξη της χρησιμότητας του παράλληλου προγραμματισμού μέσω της επίλυσης των παρακάτω ζητουμένων:

Δεδομένου δοθέντος (ως είσοδο) ενός διανύσματος **X** (μήκους *n* στοιχείων *xi | i=0…n-1*), να υπολογίζει παράλληλα σε περιβάλλον *p* επεξεργαστών και να τυπώνει στην οθόνη (ως έξοδο) τα ακόλουθα:

---

### (α) Μέση τιμή (μ)
\[
μ = \frac{x_0 + x_1 + x_2 + … + x_{n-1}}{n}
\]

### (β) Μέγιστη τιμή (m)
Για κάθε *xi* ισχύει ότι *xi ≤ m* και υπάρχει *xi* τέτοιο ώστε *xi = m*.

### (γ) Διασπορά (var)
\[
var = \frac{(x_0 - μ)^2 + (x_1 - μ)^2 + … + (x_{n-1} - μ)^2}{n}
\]

### (δ) Νέο διάνυσμα Δ
Κάθε στοιχείο *δi* ισούται με το τετράγωνο της διαφοράς του αντίστοιχου στοιχείου *xi* του διανύσματος Χ από τη μέγιστη τιμή *m* του διανύσματος:

\[
δi = (xi - m)^2
\]

---

## Περιγραφή

Χρησιμοποιούνται μόνο απλές συναρτήσεις επικοινωνίας τύπου **MPI_Send() / MPI_Recv()**, χωρίς συναρτήσεις συλλογικής επικοινωνίας.  
Ο υπολογιστικός φόρτος κατανέμεται ισομερώς στους *p* επεξεργαστές του παράλληλου περιβάλλοντος.

Κάθε επεξεργαστής λαμβάνει (κατέχει) στην τοπική του μνήμη μόνο τα δεδομένα εισόδου που χρησιμοποιεί για τοπικούς υπολογισμούς.

Τις τιμές/στοιχεία *n*, *xi* τις δίνει ο χρήστης και τις διαβάζει από την οθόνη ο επεξεργαστής **0** (ο “κεντρικός” επεξεργαστής).  
Τα αποτελέσματα συγκεντρώνονται στο τέλος στον επεξεργαστή 0 και μέσω αυτού παρουσιάζονται στο χρήστη.

Το πρόγραμμα υποστηρίζει όλα τα πλήθη διεργασιών και όλα τα μεγέθη του διανύσματος Χ (δηλαδή και για περιπτώσεις όπου *n % p ≠ 0*), εφόσον το μέγεθος του πίνακα είναι μεγαλύτερο του πλήθους των διεργασιών.

Επιπλέον, το πρόγραμμα περιλαμβάνει **μενού επιλογών** *(1. Συνέχεια – 2. Έξοδος)*, το οποίο εμφανίζεται **επαναληπτικά** μέχρι ο χρήστης να επιλέξει την έξοδο από το πρόγραμμα.
