#MPI Πρόγραμμα - AskisiI

Το παρόν MPI πρόγραμμα (σε γλώσσα C και θεωρώντας point-to-point communication) αποσκοπεί στην επίδειξη της χρησιμότητας του παράλληλου προγραμματισμού μέσω της επίλυσης των παρακάτω ζητουμένων.

Δεδομένου ενός διανύσματος Χ (μήκους n στοιχείων xi, i = 0...n-1), το πρόγραμμα υπολογίζει παράλληλα σε περιβάλλον p επεξεργαστών και εμφανίζει τα εξής:

---

###(α) Μέση τιμή (μ)

μ = (x0 + x1 + x2 + ... + x(n-1)) / n

###(β) Μέγιστη τιμή (m)

Για κάθε xi ισχύει xi ≤ m και υπάρχει τουλάχιστον ένα xi τέτοιο ώστε xi = m.

###(γ) Διασπορά (var)

var = ((x0 - μ)² + (x1 - μ)² + ... + (x(n-1) - μ)²) / n

###(δ) Νέο διάνυσμα Δ

Κάθε στοιχείο δi ισούται με το τετράγωνο της διαφοράς του αντίστοιχου στοιχείου xi από τη μέγιστη τιμή m:

δi = (xi - m)²

##Περιγραφή

Χρησιμοποιούνται μόνο απλές συναρτήσεις επικοινωνίας τύπου MPI_Send() και MPI_Recv(), χωρίς συναρτήσεις συλλογικής επικοινωνίας.
Ο υπολογιστικός φόρτος κατανέμεται ισομερώς στους p επεξεργαστές του παράλληλου περιβάλλοντος.

Κάθε επεξεργαστής κατέχει στην τοπική του μνήμη μόνο τα δεδομένα που χρειάζεται για τοπικούς υπολογισμούς.

Τις τιμές n και xi τις δίνει ο χρήστης και τις διαβάζει ο επεξεργαστής 0 (ο “κεντρικός”).
Τα αποτελέσματα συγκεντρώνονται στο τέλος στον επεξεργαστή 0 και παρουσιάζονται στο χρήστη.

Το πρόγραμμα υποστηρίζει όλα τα πλήθη διεργασιών και όλα τα μεγέθη του διανύσματος Χ (ακόμη και όταν n δεν είναι πολλαπλάσιο του p), αρκεί το μέγεθος του πίνακα να είναι μεγαλύτερο από τον αριθμό διεργασιών.

Επιπλέον, υπάρχει μενού επιλογών (1. Συνέχεια – 2. Έξοδος), το οποίο εμφανίζεται επαναληπτικά μέχρι ο χρήστης να επιλέξει έξοδο.
